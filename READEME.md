# 短链服务

本项目是一个基于 Spring Boot 实现的短链服务（Short URL Service）实现，支持短链生成、解析和访问统计。
项目详情参见[介绍文档](https://fengnkt.github.io/2025/11/01/short-url/)。

## 特性

- 短链生成与解析
- 缓存加速（Redis + Caffeine）
- 缓解缓存穿透、击穿与雪崩问题
- 异步解析日志记录与统计
- 支持自增 ID + Base62 编码生成短链

## 系统架构

短链服务由以下模块组成：

1. **短链生成服务**
    - 使用自增 ID + Base62 编码生成短链
    - 支持号段分配和线性混淆，保证唯一性且难以预测
2. **短链解析服务**
    - 通过缓存（Redis、Caffeine）加速短链查询
    - 支持布隆过滤器防止缓存穿透
    - 异步记录访问日志
3. **统计模块**
    - 记录访问次数、来源、设备信息
